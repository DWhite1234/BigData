# 1.break 配合lable标签

```java
a:	for(int i = 1;i<=10;i++){
		
			for(int j = 1;j<=10;j++){
					
				if(j==6){
					break a;
				}
				System.out.print(j+"\t");
			
			}
			
			System.out.println();
		}
可以终止a标记地方的循环,a标记只能用于break的外层循环,不能用于同级循环

注意点:
	1.如果没有lable 结束的是 break所在层循环 
	2.如果存在 lable 结束的是  lable 所在层循环 

```

# 2.continue配合lable标签

```java
l:	for(int i = 1;i<=5;i++){

			for(int j = 1;j<=5;j++){
			
				if(j  ==3){
					continue l;
				}
			
				System.out.print(j+"\t");
			}
			
			//换行
			System.out.println();
		}

continnue:
		1.如果没有lable 结束的是本次循环 继续的是continnue 所在层的下一次循环
		
		2.如果有lable  结束的是本次循环  继续的是lable 所在层循环
```

# 3.流程控制语句

```java
特殊的流程控制语句：
	break:	结束循环
	
	       只能用在循环内 或者 switch中
	
	continue: 结束本次循环 继续下一次循环
	
			只能用在循环内
		
	return: 结束方法 
	        return 下的内容 不再执行(如果有,编译会报错)
```

# 4.质数

```java
含义:只有1和本身两个约数的数是质数

优化1： 一个数的约数 除了1和本身之外 最大的约数 也就是 自身的一半  所以 
			在确定约数范围时  找到它的一半就OK
			
优化2：一个数的约数 不会单独出现在开平方的右边  查找约数范围时 只需要找到开平方数（了解）
```
